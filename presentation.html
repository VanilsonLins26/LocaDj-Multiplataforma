What is LocaDJ?

A comprehensive multi-platform application designed to connect DJs, event organizers, and party lovers in one seamless ecosystem.

Built with React Native, Expo, and Firebase.

Why We Created It

  • The Problem: Finding, hiring, and managing reliable DJs for events is a chaotic and fragmented process.
  • The Solution: A centralized hub for DJ profiles, availability, equipment management, and secure bookings.
  • The Vision: To professionalize and simplify the event industry through accessible technology.

Technologies Used

React Native

Core framework for building the cross-platform application.

Expo

Toolchain for fast development, building, and EAS deployment.

TypeScript

Ensuring robust, type-safe, and highly maintainable code.

Firebase

Backend for Authentication and real-time Database storage.

Expo Router

Modern file-based routing for seamless navigation flows.

Reanimated & Charts

Smooth animations and interactive data visualizations.
